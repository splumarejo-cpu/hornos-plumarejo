Cómo lo hacemos

Del acero crudo
al horno perfecto

Cada horno pasa por semanas de trabajo artesanal. Este es el proceso que garantiza que dure generaciones y funcione como el primer día.

01
Base y Estructura

Base metálica de acero al carbón soldada a mano, garantizando soporte estructural absoluto.

02
Piso refractario

Ladrillo certificado colocado con precisión milimétrica, diseñado para retención térmica y pizzas perfectas.

03
Cúpula napolitana

Ladrillo sobre ladrillo con Mortero Óptimo, calculada para una distribución de calor inmejorable.

04
Acabado Artesanal

Recubrimiento térmico multicapa y mosaico instalado a mano para una estética premium.
